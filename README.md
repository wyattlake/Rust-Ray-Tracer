# Rust Ray Tracer

A ray tracer written in Rust following The Ray Tracer Challenge by Jamis Buck.

The Ray Tracer currently supports:
- Primitives (Spheres, Triangles, Cubes, Cones, and Cylinders)
- Phong shading
- Patterns
- Reflection
- Refraction
- OBJ files
- Anti Aliasing
- Soft shadows
- Constructive Solid Geometry
